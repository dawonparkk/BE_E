# BE_E

## 충전소 위치와 평균 이용률/ 이용시간에 따른 패턴 데이터를 분석하여 충전소에 따른 관광 경로 추천 시스템 제공
* tmap API 이용
* 활용 데이터: 한국환경공단/한국환경공단/한국관광공사
  *한국전력공사*
    * 충전소 데이터(2018-2019)
    * 전기차 시도별 차량 등록수(2020.05 기준)
    * 충전소 리스트(20.06.30 기준)
  *한국환경공단*
    * 시도별 한국환경공단 충전이력 (2019년)
  *한국관광공사*
    * 한국 관광 100선(2018-2020)
    * 언택트 바캉스 데이터(2020)
